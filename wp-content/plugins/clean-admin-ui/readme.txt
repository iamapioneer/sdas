Easy Wordpress Menu, Remove menu items from the admin menu

== Description ==
Use this plugin to hide admin menu items like Posts, Comments, Users, etc
This makes it easier for your clients to navigate Wordpress, no distractions, leave only those options that are required.

== Installation ==
1. Upload the `clean-admin-ui` directory to the `/wp-content/plugins/` directory. 
2. Activate `Clean Admin UI` through the 'Plugins' menu in WordPress end select which items to hide.
